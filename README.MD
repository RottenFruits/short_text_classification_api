# How to use
- clone and container start.

```
git clone https://github.com/RottenFruits/short_text_classification_api.git
cd short_text_classification_api
docker-compose build
docker-compose up -d
```

- access localhost
    - http://localhost:8080/

- http://nlp.ist.i.kyoto-u.ac.jp/kuntt/
